# Follow-up Playbook

## Timing

- Use explicit timelines first: dates, deadlines, "next week", "by Friday", interview times, offer expirations, and take-home due dates.
- Use the user's precedent second: how quickly they normally reply, how formal the relationship is, and whether the search is casual or urgent.
- If there is no precedent, suggest a conservative reminder instead of asserting a fixed cadence.
- For warm intros and active recruiter conversations, prefer prompt, concise replies unless the user has signaled a slower/casual mode.
- After interviews or important calls, prepare a thank-you or recap draft when it would match the formality of the process.

## Message Shape

- Keep it short.
- Include the role/company and the last point of context.
- Make the next ask specific.
- Avoid sounding entitled to a response.

## Agent Behavior

When helping with follow-ups, inspect the application note first, infer the current stage, and draft one message that moves the conversation toward the next concrete step.
