## Notes

- Identity and logistics fields should mirror `profile.md`. If they drift, `profile.md` wins — the agent will sync.
- Reusable essays are 2–4 sentences. Long answers feel canned and don't fit modern application forms.
- For per-role placeholders, the agent generates fresh content each time, grounded in the role snapshot — don't try to pre-write them here.
